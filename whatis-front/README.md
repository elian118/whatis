## Whatis web

---

1. 구성
   - [React.JS](https://ko.react.dev/) + [Next.JS](https://nextjs.org/) + [TailwindCSS](https://tailwindcss.com/) + [DaisyUI](https://daisyui.com/)

2. 실행
   - [Node.JS 설치](https://nodejs.org/ko/download)
   - Yarn 설치(선택)
     ```shell
     npm install -g yarn
     ```
   - 프로젝트 폴더에서 의존성 설치
     ```shell
     npm install
     # yarn install # yarn 설치한 경우 대체 실행 가능
     ```
   - 터미널에서 아래 명령으로 실행
     ```shell
     npm run dev
     # yarn dev # yarn 설치한 경우 대체 실행 가능
     ```
